# MASTER · Молдова

Рабочий MVP сайта для сервиса поиска мастеров и строительных заказов в Молдове (RU/RO).

## Реализовано в приложении

Паттерны в духе **freelancer.com** (биржа: заявка → отклики → сравнение → выбор) и **kwork.ru** (каталог услуг с фикс-ценой и сроком, «что входит»).

- Главная: два пути (биржа / услуги), категории, популярные услуги, свежие заявки, «как это работает», пульс площадки.
- **Заказы** (`#/jobs`): фильтры по городу, категории, тексту; сортировка; карточка заявки с **таблицей откликов**.
- **Услуги** (`#/gigs`): каталог с фильтрами; карточка услуги; **заказ услуги** (сохраняется в кабинете заказчика).
- **Мастера** (`#/workers`): фильтры, рейтинг, портфолио, вкладки профиля; услуги мастера на профиле.
- Пошаговая **заявка** на биржу; кабинет мастера: профиль, **создание своих услуг**, отклики.
- Кабинет заказчика: заявки + **заказы услуг** отдельно.
- Отзывы, модерация, события, RU/RO; страница «Сообщения» (заглушка под чат следующего релиза).

## Запуск

Проект статический, сборка не требуется.

1. Откройте `index.html` в браузере.
2. Для полного теста пройдите сценарий:
   - Войти как заказчик -> создать заявку.
   - Войти как мастер -> отправить отклик.
   - Войти как заказчик/admin -> выбрать мастера.
   - Оставить отзыв.
   - Проверить метрики и события в `Admin moderation`.

## Тестовые роли

- Admin: создайте вход с ролью `admin` и любым телефоном.
- Worker: создайте вход с ролью `worker`.
- Client: создайте вход с ролью `client`.
