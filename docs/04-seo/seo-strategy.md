# SEO стратегия (волна 1)

## Цель на 2-6 месяц

Построить органический поток заявок за счет страниц `услуга x город x район` на двух языках.

## Шаблон URL

- RU: `/ru/services/{serviceSlug}/{citySlug}/{areaSlug}`
- RO: `/ro/servicii/{serviceSlug}/{citySlug}/{areaSlug}`

## Приоритетные города

1. Chisinau
2. Balti

## Приоритетные услуги

- electrica / elektrik
- sanitar / santehnika
- finisaje / otdelka
- acoperis / krovlya
- placare / plitka
- reparatii-mici / melkiy-remont

## Шаблон SEO-блока страницы

1. H1: Услуга + город + район.
2. 1 абзац "когда нужна услуга".
3. Ориентир цены (диапазон).
4. "Как выбрать мастера" (чек-лист из 5 пунктов).
5. CTA "Создать заявку".
6. FAQ (3-5 вопросов).

## Минимальные требования к каждой странице

- Уникальный title и meta description RU/RO.
- Уникальный H1.
- Внутренние ссылки на соседние услуги и районы.
- JSON-LD для `FAQPage` (если есть FAQ).
- Canonical + hreflang RU/RO.

## KPI SEO-волны

- Индексация 80%+ опубликованных страниц за 45 дней.
- CTR из поиска по бренд+услуга > 4%.
- Конверсия SEO-посадочных в создание заявки > 2%.
