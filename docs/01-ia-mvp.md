# IA MVP: структура сайта и меню

## 1) Карта сайта (MVP)

- `/` Главная (выбор роли + CTA).
- `/request/new` Мастер создания заявки (4 шага).
- `/jobs` Лента заявок (для мастеров).
- `/jobs/:id` Карточка заявки + отклики.
- `/workers` Каталог мастеров.
- `/workers/:id` Публичный профиль мастера.
- `/bid/new?jobId=:id` Отклик мастера.
- `/reviews` Лента отзывов.
- `/for-workers` Раздел для мастеров (о сервисе + тарифы).
- `/pricing` Тарифы PRO, пакеты откликов, поднятие профиля.
- `/auth` Вход/регистрация по SMS.
- `/account/client` Кабинет заказчика (заявки и выбранные мастера).
- `/account/worker` Кабинет мастера (анкета, отклики, статусы).
- `/admin/moderation` Админ-модерация (очередь, блокировки, заметки).
- `/legal/terms` Пользовательское соглашение.
- `/legal/privacy` Политика конфиденциальности.
- `/legal/moderation` Правила модерации.
- `/legal/reviews` Правила отзывов и споров.
- `/support` Поддержка и FAQ.

## 2) Главное меню (Header)

- `Найти мастера` -> `/workers`
- `Создать заявку` -> `/request/new`
- `Категории` -> якорь/дропдаун на главной и SEO-страницы
- `Для мастеров` -> `/for-workers`
- `Отзывы` -> `/reviews`
- `Вход` -> `/auth`
- Переключатель языка: `RU | RO`

## 3) Footer

- Города и районы: Кишинев, Бельцы + ключевые районы.
- Юридические ссылки: Terms, Privacy, Moderation, Reviews.
- Сервис: FAQ, Поддержка, Контакты.
- Для мастеров: тарифы, правила откликов, верификация.

## 4) Приоритеты разделов

- P0: `/`, `/request/new`, `/jobs`, `/jobs/:id`, `/workers/:id`, `/bid/new`, `/auth`, `/admin/moderation`, юридические базовые страницы.
- P1: `/reviews`, `/account/client`, `/account/worker`, `/pricing`, `/for-workers`.

## 5) Контент в главном экране

- Заголовок RU: "Найдите мастера для ремонта и стройки за 2 минуты".
- Заголовок RO: "Gaseste un mester pentru reparatii si constructii in 2 minute".
- Подзаголовок: "Опишите задачу — получите отклики от мастеров рядом".
- CTA 1: `Создать заявку`.
- CTA 2: `Я мастер, хочу заказы`.
