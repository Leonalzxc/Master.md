# Юридические страницы MVP

Этот раздел содержит базовые тексты для публикации на сайте.

## Файлы

- `terms.md` — пользовательское соглашение.
- `privacy.md` — политика конфиденциальности.
- `moderation-policy.md` — правила модерации и запреты.
- `reviews-policy.md` — правила отзывов и споров.
- `refund-policy.md` — политика возвратов для платных услуг мастеров.

## Важно

Тексты являются продуктовым шаблоном. Перед запуском платежей и публичного релиза рекомендуется финальная проверка юристом в юрисдикции Молдовы.
