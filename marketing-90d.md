## Phase 0: Pre-Launch Seeding (Weeks 1–4)

> Цель: 30 верифицированных мастеров ДО публичного запуска. Без мастеров = нет ликвидности.

### Канал 1: Прямой Outreach (Бесплатно)

**Скрипт WhatsApp/Viber:**
```
Привет! Я запускаю новую площадку для мастеров в Кишинёве — mastermoldova.md
Бесплатно для мастеров в первые 3 месяца.
Уже есть заявки на электрику/сантехнику/отделку.
Зарегистрироваться за 2 мин: [ссылка]
Заинтересованы?
```

**Где искать мастеров:**
- OLX.md → раздел "Услуги" → фильтр по Кишинёву/Бельцам → скрапить номера телефонов
- 999.md → аналогично
- Facebook-группы: "Ремонт в Кишиневе", "Мастера Молдовы", "Strojka MD"
- Viber-сообщества строителей/ремонтников
- Объявления на подъездах (визитки)

**Цель:** 5 мастеров/день × 6 дней = 30 за неделю

### Канал 2: Facebook Group Posts (Бесплатно)

Группы для публикации:
- "Ремонт и строительство Кишинёв" (Facebook)
- "Мастера и строители Молдовы" (Facebook)
- "Strojka MD" (Facebook/Viber)
- "Кишинёв объявления" (Viber)

Пост-шаблон:
```
🔨 Мастера Кишинёва — есть заявки на работу!
Новый сервис mastermoldova.md — заявки от заказчиков напрямую к вам.
✅ Первые 3 месяца — бесплатно
✅ Только реальные заявки с контактами
✅ Электрика, сантехника, отделка, кровля
Регистрация за 2 минуты: [ссылка]
```

### Канал 3: OLX.md / 999.md бесплатные объявления

- Разместить объявление "Ищем мастеров: электрики, сантехники, отделочники" в разделе Работа/Услуги
- Включить номер WhatsApp/Viber для входящих

## Phase 1: Soft Launch (Weeks 5–8)

> Первые реальные заявки. Invite-only неделю, потом открытый доступ.

### Канал 4: Facebook/Instagram Ads — мастера

**Аудитория:** Мужчины 25–55, Кишинёв+Бельцы, интересы: ремонт, строительство, инструменты

**Бюджет:** 50 EUR/мес (≈ 1000 MDL)

**Креативы:**
```
Ad A (мастер): 
"Устал искать клиентов? Получай заявки прямо в телефон.
Мастера в Кишинёве зарабатывают с mastermoldova.md
Регистрация бесплатная → [CTA]"

Ad B (социальное доказательство после 2 недель):
"Уже 50 мастеров в Кишинёве получают заявки на ремонт.
Присоединяйся → [CTA]"
```

### Канал 5: Facebook/Instagram Ads — заказчики

**Аудитория:** Кишинёв+Бельцы, интересы: ремонт, новоселье, IKEA, строительные магазины

**Бюджет:** 50 EUR/мес

**Креативы:**
```
Ad C (pain point):
"Ищешь мастера в Кишинёве и не знаешь, кому доверять?
Опубликуй заявку — получи 3-5 откликов от проверенных мастеров.
Бесплатно для заказчиков → [CTA]"

Ad D (speed):
"Нужен электрик сегодня в Кишинёве?
Первые отклики за 15 минут. mastermoldova.md"
```

### Канал 6: Telegram/Viber рассылка

- Войти в Telegram-чаты строителей Молдовы
- 1 пост/неделю с новыми заявками (анонсировать актуальные)
- Не спамить — быть полезным участником

### Канал 7: PR / Нетворкинг

- Написать в Bizlaw.md, Privesc.md, IPN.md, Locals.md (новостные порталы Молдовы)
- Питч: "Первая биржа мастеров в Молдове, запуск в Кишинёве"
- Цель: 1–2 бесплатные публикации

## Phase 2: Growth (Weeks 9–13)

### Канал 8: SEO (органика, долгосрочно)

**Страницы волны 1 (из seo-pages-wave1.csv):**
- 6 услуг × 2 города × 5 районов = 60 страниц RU
- 60 страниц RO = 120 страниц итого
- Срок индексации: 30–45 дней после публикации

**KPI SEO:**
- Индексация ≥ 80% за 45 дней
- Первые позиции по "[услуга] [район] Кишинёв" через 60–90 дней
- CTR из поиска > 4%

### Канал 9: Реферальная программа для мастеров

```
"Приведи друга-мастера → получите оба по 10 бесплатных откликов"
Механика: уникальный реф-код в кабинете мастера
```

### Канал 10: Контент-маркетинг

Статьи для блога (MDX), 2–4 в месяц:
- "Сколько стоит замена проводки в квартире в Кишинёве (2026)"
- "Как выбрать сантехника: 7 вопросов перед наймом"
- "Отделка квартиры под ключ: цены в Молдове"

```
RU: "Заявки от заказчиков — прямо к тебе. Без посредников. Бесплатно 3 месяца."
RO: "Comenzi de la clienți — direct la tine. Fără intermediari. Gratuit 3 luni."

```

RO: "Găsește un meșter în 2 minute. Primești 3–5 oferte. Alegi cel mai bun."

## Anti-Churn: удержание мастеров

- Email/SMS если мастер не открывал приложение 7 дней: "Новые заявки по вашим категориям"
- Уведомление при новой заявке по категории мастера (Viber/Telegram)
- После 1-го выбора: поздравление + "Расскажи друзьям-мастерам"
- После 3-го выбора: предложение PRO со скидкой 20%
