### CI/CD Pipeline

```
PR открыт
  → lint + type-check (2 мин)
  → unit tests Vitest (1 мин)
  → Vercel Preview deploy (3 мин)
  → Cypress E2E на Preview URL (5 мин)
  → Code review (фаундер)
  ↓
Merge в main
  → Vercel Production deploy (2 мин)
  → Smoke test (1 Cypress spec)
  → Sentry release создаётся
```

### Обязательные E2E тесты

| Сценарий | Шаги | Критерий |
|----------|------|---------|
| client_happy_path | Форма → OTP → публикация | job появился в DB |
| worker_happy_path | Регистрация → анкета → отклик | bid в DB, credits-- |
| moderation | Блокировка → попытка входа | 403 |
| payment | Покупка bid_pack | credits++ |
| anti-fraud OTP | 4 попытки OTP → блокировка | 429 |

### OWASP Top-10 Matrix

| Угроза | Митигация |
|--------|----------|
| A01 Broken Access Control | RLS на всех таблицах + middleware role-check |
| A02 Crypto Failures | httpOnly cookie, SHA-256 для phone_hash, HTTPS forced |
| A03 Injection | Supabase SDK (parameterized), Zod валидация на входе |
| A04 Insecure Design | Rate-limit OTP, bid_credits, blacklist |
| A05 Security Misconfiguration | Security headers, CSP, RLS default deny |
| A07 Identification Failures | OTP + rate-limit, нет паролей |
| A08 Software & Data Integrity | Payment webhook HMAC, idempotency key |
| A09 Logging Failures | Sentry, moderation_log append-only |
