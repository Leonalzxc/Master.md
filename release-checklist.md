## 🔴 БЛОК 4: Безопасность (P0)

- [ ] Security headers: X-Frame-Options, X-Content-Type-Options, CSP

- [ ] XSS: все пользовательские данные escaped при рендере
- [ ] Phone-номера в DB: только E.164 формат, валидация на бэкенде
- [ ] Blacklist хранит SHA-256(phone), не сырые номера
- [ ] Admin-роут `/admin/*` доступен только role=admin (RLS + middleware)
- [ ] Payment webhook проверяет HMAC-подпись шлюза
- [ ] Rate-limit на все mutation endpoints
- [ ] Логи модерации: нельзя редактировать (append-only)
- [ ] Sentry подключён и ловит ошибки (протестировано)

## Go/No-Go решение

### Go (можно запускать):
- [ ] Все P0 чеклисты ✅
- [ ] Не менее 20 верифицированных мастеров онборжены
- [ ] Хотя бы 3 тестовые заявки прошли полный цикл (заявка → отклик → выбор → отзыв)
- [ ] Юрлицо зарегистрировано
- [ ] Поддержка доступна (хотя бы WhatsApp)

### No-Go (остановить):
- Реальный OTP не работает
- Платёжный шлюз без договора
- Admin-панель доступна анонимусам
- localStorage вместо Supabase в production
